## Current Research

### Hyperspectral Estimation of Crop Biochemical Properties

My master's thesis investigates the use of spaceborne hyperspectral imagery and machine learning to estimate crop biochemical properties under different agricultural management treatments.

The research focuses on estimating:

- Canopy Chlorophyll Content (CCC)
- Canopy Nitrogen Content (CNC)

using hyperspectral data from **EnMAP and PRISMA satellites**.

### Methodological Workflow

The analysis pipeline includes:

1. Hyperspectral preprocessing and geometric correction  
2. Spectral feature engineering and band selection  
3. Machine learning model development  
4. Model validation and performance evaluation  
5. Spatial prediction mapping  

Code for this research is currently private while thesis work and publications are under preparation.
